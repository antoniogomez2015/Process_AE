	Zaxwerks Easy Glow:
	___________________


	_éste plugins, nos permitirá crear el efecto de glow, ó resplandor a través, de diferentes criterios y

	parámetros lo cuál, hace extendible y versátil el efecto glow	



	Workflow:
	_________

	_para lo cuál, lo primero será tener creada una composición de trabajo así, como al menos una layer, sobre

	la cuál, vamos ha aplicar el sistema	



	Easy Glow:
	__________

	_effects pressets:

		_


	_target:

		_éstas opciones nos permiten generar el sistema de glow, a través, de ciertos atributos de

		la imágen ó layer, sobre la cuál, se generará el sistema


		_mask:

			_éstas opciones nos permiten seleccionar el tipo de criterio de acción ha realizar

			por el sistema


			_de ésta manera, podremos enmarcaras el sistema de transiciones de degradado 

			texturizado ó no texturizado a través, de diferentes métodos como por ejemplo; a

			través, del mismo canal alpha, de la respectiva layer, como también, a través,

			de un "cuadrado interno", ó "circulo interno", del sistema		

			
			_none:

				_ésta opción desactiva las opciones de enmascarado para el sistema generado


			_layer alpha:

				_ésta opción activa el enmascarado sólo si la layer, dispone de algún canal 

				alpha, a través, del cuál, podrá hacer el efecto colateral de 

				enmarcaramiento en el perímetro de las partes en alpha, por medio de las

				opsiones de "feather", y demás opciones de "mask", para generar así, a 

				través, de éstos el efecto de glow

			

			_layer brightness:

				_ésta opción realizará el enmascarado sobre las zonas con más brillo la 

				imágen ó layer, sobre la cuáles hará el efecto glow


			_layer brightness+alpha:

				_ésta opción es idéntica a la opción anterior más éste incluirá en éste 

				caso el alpha, que tenga la respectiva imágen


					
		_bright cutoff:

			_ésta opción limita la cantidad de brillo detectado sobre la respectiva imágen para

			el proceso de inmascarado ó visualización del glow para las opciones

			de "mask/layer brightness", y demás opciones


		_dark cutoff:

			_ésta opción hará variaciones sobre la cantidad y tipo detecta sobre la respectiva 

			imágen 	para el proceso de inmascarado ó visualización del glow, para las 

			opciones de "mask/layer brightness", y demás opciones


		_show target:

			_ésta opción nostrará de forma plana a través, del color seleccionado en la opción

			de "custom color", para que así, podamos ver las áreas de acción que tomará el 

			sistema para generar el efecto glow



	_glow style:

		_glow un front:

			_ésta opción creará el sistema glow, sobre el área muestreada en el paso anterior


		_in glow:

			_ésta opción creará el sistema en el área perimetral hacia adentro de cada "islan",

			de toda el área mostrada en la sección "show target"


		_out glow:

			_ésta opción creará el sistema en el área perimetral de cada "islan", de toda el

			área mostrada en la sección "show target"


	_glow size:

		_ésta opción aumentará ó disminuirá la extensión y difusión del efecto glow, sobre la 

		imágen y área "target"


	_softness:

		_ésta opción extenderá ó contraerá el efecto glow, sobre la imágen así, como también, su

		intensidad ó posterizado resultante ó colateral


	_glow color from:

		_custom color:

			_ésta opción generará el sistema glow, bajo los parámetros indicados a través, de

			un color que podemos determinar e incluso animar


		_this layer:

			_ésta opción generará el sistema de glow, bajo los parámetros indicados a través,

			de los colores referenciales de la misma imágen sobre la cuál, hemos generado el

			sistema, generando así, diferentes resultados en diferentes grados


	_custom color:

		_ésta opción nos permite indicar ó seleccionar un color para generar el sistema



	_online documentation:

		_ésta opción nos permite redirecciorarnos ha un enlace externo del fabricante del plugins,

		para una mayor documentación de detalles
