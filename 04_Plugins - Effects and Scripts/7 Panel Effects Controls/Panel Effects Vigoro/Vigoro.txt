panel Effects Controls / Vigoro


	Vigoro:


	-este nos permite crear efectos de "strands", a partir de formas geométricas básicas generadas por el 

	sistema


	_así, de ésta manera, a través, de por ejemplo de círculos, hexagonos y otros podremos a partir de éstos

	obtener strands, ó instancias relacionadas con la shape, ó forma geométrica original


	_para que así, éstas instancias segan y compartan comportamientos con la forma original


	_de ésta manera, podremos crear variaciones animadas en el tiempo así, como anomalías de comportamientos

	así, como de forma, lo cuál, permitirá animaciones coplejas así, como el constructivismo visual lo cuál,

	es especialmente útil para motion graphycs, avanzado


	
	Workflow:
	_________

	_primero debemos de tener creada una composición de trabajo en la cuál, no recomendable que ésta composición

	sea únicamente para el sistema de "Vigoro", y ésta nueva composición no tenga layers, creadas previamente ya

	que el sistema generará todo el sistema de layers, y demás elementos por si misma



	Vigoro:
	_______


		_Vigoro Comp Name:

			_ésta opción nos permitirá renombrar las layers, generadas por el sistema


		_No of Layers:

			_ésta opción nos pemritirá indicar la cantidad de strands, ó instancias generadas por el 

			sistema


		_No Cut Effects (Enables 3D Shadows):

			_ésta opción nos permitirá generar el efecto de cortes en las layers, generadas por le 

			sistema lo cuál, será opcional


		_3D Comp:

			_ésta opción nos permitirá activar las opciones de 3d, para las layers, generadas por el

			sistema


		_Camara:

			_ésta opción nos permitirá crear una cámara para el sistema para controlar así, las 

			layers, 3d, generadas por el sistema


		_Light:

			_ésta opción nos permitirá crear luces para el sistema para iluminar así, las 

			layers, 3d, generadas por el sistema



		_Create Comp:

			_ésta opción nos permitirá generar el sistema


		_Setings:

			_éstas opciones nos permitirán configurar los criterios para la generación del sistema


		_Add Text Layer:

			_ésta opción nos permitirá agregar layers, de texto al sistema lo cuál, lo hace 

			extensible


		_Convert Layer:

			_ésta opción nos permitirá agregar layers, de forma  personalizadas al sistema de vigoro,

			ya generado previamente


		_Bake/Free Comp:

			_ésta opción nos permitirá hacer un volcado de las layers, para así, cada uno de éstas

			tenga animación propia en relación a la animación ó simulación generada por el sistema


			_de ésta manera, en las opciones de volcado podremos filtrar que deseamos volcar, 

			pudiéndo volcar efectos, transformaciones, layers, de texto, otros



	Crear el Sistema:
	_________________

	_para generar el sistema debemos de seleccionar previamente la respectiva composición de trabamo en el

	panel proyecto


	NOTA: ésto es así, silo seleccionamos en las opciones "Settings", del plugins, lo cuál, hará que el 

	hacer éste paso Vigoro tome ó emule las características de la composición selección más si no se 

	selecciona ésta opción vigoro, generará el sistema son la necesidad de seleccionar nada en el panel 

	proyecto


	_para posteriormente ir al panel del plugins, Vigoro, y hacer mouse left click, sober el botón de

	"Create Comp", para que así, el sistema sea generado en relación a los parámetros indicados ó 

	seleccionados



	Vigoro System:
	______________

	_luego de haber generado el sistema con vigoro, éste habrá creado una nueva composición con el nombre

	que le indicamos en el campo "Vigoro Comp Name", del plugins, por lo general ésta composición tendrá

	el nombre de "Vigoro"


	_ésta nueva composición se compone de las siguientes las layers, las cuáles, tienen la siguiente

	función:



		_Controller:

			_en ésta layer, se encuéntran todos los controles y funcionalismos del sistema ha modo 

			de efecto los cuáles, podremos controlar en el panel "Effects Controls", shortcut "F3"



		_Fill Color:

			_layer, interna del sistema


		_Stroke Color:

			_layer, interna del sistema


		_VVVVV:

			_layer, interna del sistema



		_Vigoro Shape 1,2,3,4,5,...:
		
			_éstas layers, representan la la forma base e instancias ó strands, del sistema


		_^^^^^:

			_layer, interna del sistema
